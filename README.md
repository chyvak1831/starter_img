#### ğŸ”§ Setup
1. Install  [Requirements](#requirements)
2. Install gulp globally (if is not installed yet) - do it once
    ```bash
    npm i --global gulp-cli
    ```
3. Go to theme folder and run cmd/terminal and install packages
    ```bash
    npm i
    ```
